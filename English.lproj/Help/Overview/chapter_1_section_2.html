Specifying files to analyze.

There are several ways to open an executable file in otx. You can drop a file onto the main window, or onto otx's icon either in the Finder or Dock. You can also use the "Open" command in otx's File menu. If you drop an application package(.app) onto otx, it will attempt to open the application's main executable file. In some cases, the application package and executable file have different names, and you must locate the executable file manually (Show Package Contents) and open it directly. Similarly, if you want to open secondary executable files inside an application package, such as frameworks, these must be located and opened manually.
